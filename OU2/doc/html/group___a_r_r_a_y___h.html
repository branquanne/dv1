Detailed Description

Declarations used to represent the abstract data type Array.

A module that contains the declarations of the struct Array and functions used to create and manipulate a 2D array. The module declarations are used by the files array.c and main.c.

Array * array_create ( int n1,
int n2 )

Create and return a zero array of a user-specified size.

Create and return a zero array of size n1 rows by n2 columns. The function allocates memory for the array and initializes all elements to zero.

Parameters
n1The number of rows.
n2The number of columns.
Returns
A pointer to the created array.
See also
array_destroy()
Note
The function does not check if the memory allocation is successful.
User will have to manually call array_destroy() in order to free the memory allocated for the array.
